C1 - Assista a video-aula:

A estratégia apresentada é uma adaptação de POLYA (1977) e STEWART (2006), para a resolução de problemas sobre taxas relacionadas:

1º) Leia cuidadosamente o problema proposto;

2º) Se possível ou necessário elabore um diagrama (sempre que possível);

3º) Observe e transcreva os dados os quais você possui (Utilizando as notações matemáticas adequadas).

4º) Verifique o dado procurado ou o que você deseja encontrar.

5º) Tente achar uma equação que relacione os dados que você possui.

6º) Aplique a REGRA DA CADEIA para relacionar os dados.(Sempre que necessário).

7º) Substitua as informações dadas. Caso falte alguma volte e analise cuidadosamente como poderá encontrá-la.

8º) Resolva a substituição e interprete a resposta encontrada para analisar se está é plausível.
